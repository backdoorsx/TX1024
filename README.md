# TX1024 - Arduino FPV Drone transmitter

Arduino + NRF24 radio receiver 100mW:
  - 4x analog channel, 0-1023 remap to 1000-2000
  - 6x AUX channel, bool
  - menu check radio
  - menu analog calibration
  - menu check baterry
  - menu settings analog and mode

```
Define SPI pins:

  D52 - SCK
  D50 - MISO
  D51 - MOSI
  D48 - SCN -------+
  D49 - CE ----+   |
               |   |
    RF24 radio(49, 48);
```
